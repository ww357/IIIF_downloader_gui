IIIF image downloader for highest-resolution lossless TIFFs

<u>instructions:</u>

Download packages: 
```bash
pip install requests pillow pyinstaller
```

Build application:
```bash
pyinstaller --onefile --windowed --name "iiif_downloader_gui" iiif_downloader_gui.py
```

Run dist/iiif_downloader_gui
